## Jukebox-Playlist


### Start Client
To start the client use the command:
```bash
npm run serve
```
in the client folder

### Start server
To start the server use the command:
```bash
php server run 
```

### Client config
##### jukebox-server.js
This config file is used to make a connection with the server.

### Server config
##### server.php
This config is used for Http headers and logging/token settings.

##### sql.ini
This config file is used to make a msql connection with your database
